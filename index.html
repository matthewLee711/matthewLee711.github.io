Zombie Tube

Over 325 million tennis balls are thrown away each year. The main reason these tennis balls end up in landfills is because the balls lose their pressure over time and eventually become unusable. Zombie Tube tackles this issue by using a high pressure chamber to force air back into tennis balls, eventually bringing them back to life! If you want to contribute reducing tennis ball waste, feel free to go checkout www.savethetennisballs.com

802.15.4 Linear Network Coding

The internet of things has seen huge growth and especially change over the past few years. The biggest ongoing change is the switch from several high powered nodes to many low powered nodes, which have little computation power. My team's reasearch focused on whether or not if it is possible to implement Linear Network Coding on the 802.15.4 protocol for improved packet transmission.

We ultimately found it is a viable to use LNC! We implemented Galois Algorithm and performed Gaussian Elimination for packet combination. Using frame slotting as an identifier for time, we saw about a 23% decrease in slotting. This improved to 25% when the packet destination header byte was removed.

In conclusion, if the ZigBee chipset or other 802.15.4 chipsets implement LNC, this will show drastic changes to IoT mesh topologies. Fewer nodes would be needed and the cost of IoT technologies would decrease in price.

Neural Networks with CUDA

An Neural Network is an information processing method that was inspired by the way biological nervous systems function, such as the brain, to process information. It is composed of a large number of highly interconnected processing elements (neurons) working in unison to solve specific problems through adaptive learning.

Titan X CUDA cores are utilized for convolution, pooling, and normalization of the neural network topology. This is important to parsing 5 gigabytes of text and creating activation layers. The main objective of this project is to establish a better understanding of machine learning and its usefulness for adapting to and asserting conclusions to data.

HLT Search Engine

Data, it's all around us! Wikibooks is a subsidiary website of Wikipedia where they host free books. I created a C++ program which parsed through over 500,000 pages of text and stored it within an hashed AVL chained table so it would allow quick access. The multiquery search uses a TF/IDF statistic to idenify the importance of a word or phrase to document inside the corpus and would bring up search results based on it.
